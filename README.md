온종일 컴퍼니의 테라헤르츠 풋 디바이스 제품 소개 랜딩페이지입니다.

## 🔗 라이브 페이지

GitHub Pages를 통해 배포됩니다.  
`https://<your-username>.github.io/<repo-name>/`

## 📁 파일 구성

```
/
└── index.html     # 메인 랜딩페이지 (단일 파일)
```

## 🛠 GitHub Pages 배포 방법

1. 이 저장소를 GitHub에 업로드합니다.
2. 저장소 **Settings** → **Pages** 탭으로 이동합니다.
3. **Source** 를 `Deploy from a branch` 로 설정합니다.
4. **Branch** 를 `main` / `(root)` 로 선택 후 저장합니다.
5. 잠시 후 위의 URL로 페이지가 활성화됩니다.
